# 🌱 AGRITECH – AI Chatbot for Farmers

AGRITECH is an AI-powered agriculture chatbot designed to assist farmers with crop guidance, pest control, fertilizers, irrigation methods, and farming techniques.

## 🚀 Features
- AI-powered agriculture assistant using Gemini API
- Telugu voice recognition support
- Voice-based response system
- Chat history management
- Agriculture marketplace redirection
- Built using Streamlit web framework

## 🛠 Tech Stack
- Python
- Streamlit
- Google Gemini API
- SpeechRecognition
- gTTS (Text-to-Speech)

## 🔐 Setup Instructions
1. Install dependencies:
   pip install -r requirements.txt
2. Set your Gemini API key as environment variable.
3. Run:
   streamlit run app.py
